

## git branch 查看本地分支

## git branch  -a 查看远程分支

## git branch develop 创建本地分支

## git checkout develop 切换分支

## git checkout -b develop 创建并切换分支

## git fetch origin develop  ，git merge origin/develop 获取develop分支代码



## git push -u origin develop 创建远程分支并提交到本分支的远程分支

## git checkout -b develop origin/develop 切换到远程该分支

## git checkout -b feature develop 基于develop分支创建新功能分支
 
## git branch -d develop 删除develop分支
